# JJK Character Showcase

An interactive character showcase experience featuring Jujutsu Kaisen characters with immersive visuals, audio, and effects.

## Features

- **Dynamic Character Selection** - Select characters to reveal their scenes
- **Custom Cursor System** - Character-themed cursors with unique effects
- **Parallax 3D Cards** - Floating cards that follow mouse movement
- **Screen Shake Effects** - Domain Expansion style vibrations for powerful characters
- **Audio Quotes** - Character-specific voice lines on selection
- **Visual Effects** - Blood, Frost, and Wave motion text effects

## Tech Stack

- React + TypeScript
- Framer Motion
- Tailwind CSS
- Vite
